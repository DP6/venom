## Syntax Indentation
Use [JsFormat](https://github.com/jdc0589/JsFormat) with the following settings

```json
{
	"indent_with_tabs": true,
	"preserve_newlines": true,
	"jslint_happy": true,
	"keep_function_indentation": false,
	"break_chained_methods": false
}
```
